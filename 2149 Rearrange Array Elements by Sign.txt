Intuition
We will maintain pointers.
First pointer will be used to place positive integers on even indexes.
Second pointer will be used to place negative integers on odd indexed.

Approach
We will create a new array with length same as nums. And place 'pos' pointer to 0 and 'neg' pointer to 1 to place intergers.
We will start iterating through nums array.
Whenever we encounter a positive number we place that in new array at position of 'pos' and increment pos by 2.
Whenever we encounter a negative number we place that in new array at position of 'neg' and increment neg by 2.
Once the whole array is traversed, we will return the new array.

Complexity
Time complexity:
O(n) as we traverse the whole array.
Space complexity:
O(n) as we creating a new array to store elements.
